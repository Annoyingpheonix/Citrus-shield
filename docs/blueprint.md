# **App Name**: Citrus Shield

## Core Features:

- Secure Tunnel Creation: Establish an encrypted connection to a VPN server.
- Server Selection: Allow users to choose a VPN server location from a list of available servers.
- Kill Switch: Implement a kill switch feature to automatically disconnect from the internet if the VPN connection drops.
- IP Address Masking: Ensure that the user's IP address is hidden and replaced with the VPN server's IP address.
- Data Encryption: Encrypt all data transmitted through the VPN tunnel to protect user privacy.
- Connection Logging: Enable or disable connection logging for troubleshooting.
- Optimal server location suggestions: Use AI to suggest the optimal server based on speed tests to available servers.

## Style Guidelines:

- Primary color: Vivid orange (#FFA500) to convey security and energy.
- Background color: Light, desaturated orange (#F2E6D9).
- Accent color: Yellow (#FFD447) to highlight key features.
- Body and headline font: 'Inter', a grotesque-style sans-serif, provides a clean, modern look for both headlines and body text.
- Use minimalist icons to represent server locations, security features, and settings.
- Create a clean, intuitive layout with a focus on ease of use and quick access to key features.
- Incorporate subtle animations to provide visual feedback and enhance the user experience, such as connection status indicators.
